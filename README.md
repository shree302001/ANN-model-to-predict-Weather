# Weather Analysis and Prediction Using Artificial Neural Networks (ANN)

**Published research project.** Predictive models for analyzing and forecasting complex meteorological datasets using Artificial Neural Networks — establishing the foundational expertise in Pattern Recognition and Predictive Analytics that drives current enterprise NLP and AI work.

## Overview

Traditional weather forecasting relies on numerical methods that are computationally expensive and struggle with non-linear atmospheric patterns. This project applies ANN-based deep learning to learn directly from historical meteorological data and produce accurate multi-variable weather forecasts.

## Tech Stack

| Tool | Purpose |
|---|---|
| Python 3 | Core language |
| TensorFlow / Keras | Neural network model |
| NumPy + Pandas | Data processing |
| Matplotlib + Seaborn | Visualisation |
| scikit-learn | Preprocessing, evaluation |

## Model Architecture

- **Input layer** — meteorological features (temperature, humidity, pressure, wind speed, etc.)
- **Hidden layers** — fully connected with ReLU activations + dropout for regularisation
- **Output layer** — predicted weather variable(s) for the forecast horizon
- **Optimiser** — Adam | **Loss** — MSE

## Results

- Demonstrated strong predictive accuracy on historical meteorological test sets
- Outperformed baseline statistical models on non-linear weather patterns
- Established core expertise in neural network design and time-series forecasting
